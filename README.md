# leanote desktop app

Use node-webkit to create leanote desktop app

## How to develop it

### Install node-webkit

See https://github.com/nwjs/nwjs


### How to run it

Download this project, and run

````
$> cd PATH-TO-LEANOTE-DESKTOP-APP/src
$> PATH-TO-NODE-WEBKIT/nw .
```

### how to build it
```
cd PATH-TO-LEANOTE-DESKTOP-APP
$> npm install
$> gulp
```
