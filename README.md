# Leanote Desktop App

[![Join the chat at https://gitter.im/leanote/desktop-app](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/leanote/desktop-app?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

Use Electron(atom-shell) to create leanote desktop app.

![preview.png](preview.png "")

## Download
Please see http://app.leanote.com

## How to develop it

### 1. Install Electron

See https://github.com/atom/electron


### 2. How to run it

Download this project, and run

````
$> cd PATH-TO-LEANOTE-DESKTOP-APP/src
$> Electron .
```
